
- version 3.0

################
# DEPENDENCIES #
################

- Cmake 3.10 minimum
- Opencv 3.4 minimum
- Ceres 1.12 minimum (optional)
- Eigen 3 minimum

Ubuntu 16.04LTS : sudo apt-get install cmake ceres eigen3 opencv

################
# INSTALLATION #
################

1. extract library folder
2. mkdir build
3. cd build
4. cmake ..
5. make
6. sudo make install (optional)
7. make doc (optional for generating documentation)
