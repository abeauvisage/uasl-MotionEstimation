
- version 3.0

################
# DEPENDENCIES #
################

- Opencv 3.4 minimum
- Ceres 1.12 minimum (optional)
- Eigen 3 minimum

################
# INSTALLATION #
################

1. extract library folder
2. $ mkdir build
3. $ cd build
4. $ cmake ..
5. $ make
6. $ sudo make install
7. $ make doc (optional for generating documentation)
